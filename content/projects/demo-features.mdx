Welcome to the feature showcase! This project demonstrates the new markdown components we just added to the site. The layout now features a sticky **Table of Contents** on the right side of the screen, which automatically extracts headings from the document.

## Syntax Highlighting

Code blocks are automatically styled with the `github-dark` theme. It parses language tags perfectly.

```tsx
// Example of a React component
import { useState } from "react";

export function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      Clicked {count} times
    </button>
  );
}
```
