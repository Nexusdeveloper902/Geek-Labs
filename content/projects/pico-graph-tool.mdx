This project turns a standard Raspberry Pi Pico into a high-performance graphing and visualization tool. Using an I2C OLED display and an analog joystick, users can view 3D wireframes or plot live sensor data.

## Features
- **High-speed rendering**: Custom rasterization algorithm optimized for the RP2040.
- **Joystick Input**: Navigate menus, zoom into data, and rotate 3D objects in real time.
- **Multicore**: Core 0 handles the display rendering, while Core 1 processes incoming sensor data.

## The Setup
I am using a standard breadboard setup with a 128x64 OLED and a dual-axis joystick module.

The code is fully open-source and written in C++ using the Pico SDK.
